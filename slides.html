Code highlighting

# Variables and expressions. a = 10 b = 3 * a + 2 printf("%d %d\n", a, b); # Type is dynamic. b = "A string" c = 'Another String' print b + " and " + c + "\n"

You just need to add the "hljs" attribute to your element: <div hljs>your code here</div>

 ##Markdown content write your slide in markdown if you like it - list item **one** - list item *two* Add the "markdown" attribute to the slide element `<slide markdown>`

Auto fitting images

Add the "bg-img" attribute to your slide element:
<slide bg-img="path/to/image">
without worrying about the image size.

You can add credits too (with autolink)
<slide bg.img="..." credits="...">
